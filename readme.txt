== Description ==

This plugin add additional widget area that can be slide up and down.
Very useful when your site have lot of information to show but doesnt have enough space.

Note: This plugin only supports Genesis child themes.

== Installation ==

1. Upload the entire `Genesis Widget Toggle` folder to the `/wp-content/plugins/` directory
1. DO NOT change the name of the Genesis Widget Toggle` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag and drop 'Genesis Widget Toggle' into active sidebar
